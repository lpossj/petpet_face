# 更新日志

本项目遵循语义化版本号。

## v0.0.2 - 2026-07-15

### 修复

- 修复实时摄像头预览 FPS 过低的问题，减少人脸检测对预览刷新率的阻塞。
- 修复底部状态栏遇到长录制路径时换行并显示不完整的问题。

### 调整

- 调整手部动画的显示大小，使覆盖区域更自然。
- 调整人脸形变特效的作用范围和大小。

### 新增

- 新增独立设备设置窗口，可选择摄像头和麦克风。
- 新增麦克风录音；停止录制后自动将音频与特效视频合并为 MP4。
- 新增“不录音”选项，可继续生成无声 MP4。
- 首次启动默认使用系统默认麦克风录音；只有主动选择“不录音”才关闭音频。
- 新增摄像头与录制快捷键自定义，支持单键及 `Ctrl`、`Shift`、`Alt` 组合键。
- 新增设置持久化，自动保存设备和快捷键选择。
- 新增“下载 GPU 组件”按钮，由程序直接下载和安装组件。
- 安装器不再自动下载大型 GPU 组件，避免网络超时导致安装失败。
- 修复安装器自动下载 GPU 组件时可能出现的 12002 超时问题。
- GPU 组件按钮改为 v1.0.0 ZIP 直链；下载完成后自动定位组件并解压到程序的 `gpu` 文件夹。
- GPU 组件下载改为内置 WinHTTP 直连 HTTPS，新增进度条和百分比；“停止下载”、取消设置或关闭设置窗口均可中止传输并清理临时文件。
- 精简发布目录，仅保留主程序 `PetPetFace.exe`，移除三个重复命名的可执行文件。

## v0.0.1 - 2026-07-15

- 首个公开版本。
- 支持本地视频处理、摄像头实时特效、CPU 与 NVIDIA CUDA 推理。
- 支持人脸跟踪、平滑和短时丢失补偿。
